# Demo placeholders

This folder is for **small, safe placeholder assets** only — not for CCTV archives or large video files.

## Allowed here

- Tiny static images or SVG illustrations for UI placeholders
- Short text manifests describing external demo links
- `.gitkeep` to preserve the folder structure

## Not allowed here

- Raw CCTV or municipal camera footage
- Large stock video clips (unless license explicitly allows **redistribution in this repo** — rare)
- Files with identifiable faces or number plates
- Downloaded research datasets or model weights

## Preferred patterns

| Need | Approach |
|------|----------|
| Live demo video | Host on YouTube/Vercel Blob with documented license |
| Stock traffic clip | Link externally in README or `DATA_SOURCES.md` |
| App walkthrough | Use [live demo](https://civic-ai-platform-three.vercel.app) or `docs/screenshots/` |
| Real pilot clips | Encrypted storage outside Git — see `DATA_SOURCES.md` |

See [DATA_SOURCES.md](../../DATA_SOURCES.md) and [docs/data/README.md](../../docs/data/README.md).
