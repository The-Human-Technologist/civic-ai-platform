# Sample data (synthetic)

All data in this directory is **fake demo data** for the open-source MVP.

## What is synthetic

| File | Contents |
|------|----------|
| `locations.ts` | Public place names around Barasat, North 24 Parganas — coordinates are approximate demo values |
| `pilot-proposal.ts` | Fictional pilot proposal copy for demonstration |
| `event-store.tsx` | Client-side store; events generated by `mock-ai/processor.ts` |
| `sample-events.json` | Reference stub only — live seed data is in `processor.ts` |

## Event IDs

Format: `EVT-BRS-24xx` — **not** real government case numbers.

## Camera labels

Examples: `BMC-CAM-01`, `TP-CAM-02` — **fictional** device identifiers.

## No personal data

- No real names, phone numbers, Aadhaar, or vehicle registration numbers
- No real faces or number plates in the repository
- Reviewer field uses placeholder `"Pilot Reviewer"`

## Adding demo data

When adding locations or events:

1. Use **public road/junction names** only
2. Add a comment: `// SYNTHETIC DEMO DATA`
3. Do not use real citizen complaints or leaked CCTV metadata
4. See [PRIVACY.md](../../PRIVACY.md) and [CONTRIBUTING.md](../../CONTRIBUTING.md)
